# 一年級互動練習

這個 repo 收錄給國小一年級使用的互動練習網頁。

## 已有練習

### 一年級南一國語期末2互動練習

範圍：

- 圈出正確的字
- 選出正確答案
- 填詞選空
- 重組句子
- 照樣造句
- 改錯字
- 標點符號
- 量詞練習
- 閱讀理解

### 小一數學第六至第九單元互動測驗

範圍：

- 第六單元：100以內的數
- 第七單元：認識錢幣
- 第八單元：二位數的加及二位數的減
- 第九單元：幾月幾日星期幾

功能：

- 可單選或複選單元
- 複選時自動合併成綜合測驗卷
- A/B 兩套題組切換
- 圖像化題型：百數表、數線、錢幣、價格表、直式框、月曆
- 可直接列印

開啟 `index.html` 即可選擇練習。
